    7. A callback routine is a function that is passed as an argument to another function and is executed at a later time, usually in response to an event or condition. In the context of sorting, a callback function is often used to define the sorting criteria. For example, we might pass a callback function to the `sort` method of an array, which will be called for each pair of elements being compared. The callback function should return a negative number if the first element should come before the second, a positive number if it should come after, and zero if they are equal.
